Final Thesis

Driver fatigue is a significant cause of road accidents. This project presents a real-time driver drowsiness alert system using Raspberry Pi, sensors, and computer vision (OpenCV with Python).

The system monitors the driver’s face and eyes through a camera, detecting eye closure. If the driver’s eyes remain closed for more than 3 seconds while the vehicle is moving, a buzzer alerts the driver.

OpenCV is used to detect facial features and track eye movements, providing a cost-effective solution for reducing drowsiness-related accidents.
